ProPair - Assets

Placez vos fichiers ici :

1. LOGO
   - logo.svg (ou logo.png) - Logo principal ProPair
   - logo-white.svg - Version blanche du logo (si nécessaire)

2. FAVICON
   - Le favicon.ico doit être placé dans /public/favicon.ico
   - Vous pouvez aussi ajouter :
     - favicon-16x16.png
     - favicon-32x32.png
     - apple-touch-icon.png (180x180)

Structure recommandée :
/public
  ├── favicon.ico
  ├── apple-touch-icon.png
  └── images/
      ├── logo.svg
      └── logo-white.svg
